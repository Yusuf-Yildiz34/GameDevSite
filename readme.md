# Stajın Şifreleri

**Stajın Şifreleri**, üniversite öğrencilerine staj süreci hakkında rehberlik etmek
amacıyla düzenlenen bilgilendirici bir etkinliktir.

## Etkinlik Hakkında
Bu etkinlikte;
- Staj başvurularında nelere dikkat edilmesi gerektiği
- CV hazırlama süreci
- LinkedIn kullanımının önemi
- Firma beklentileri

gibi konular ele alınacaktır.

## Etkinlik Bilgileri
- **Tarih:** 2 subat 2026  
- **Saat:** 13.00 – 14.30  
- **Yer:** Ziraat fakultesi E43 numarali salon

## Kimler Katılmalı?
- Staj arayan üniversite öğrencileri
- 1., 2. sınıf öğrencileri
- Kariyerini erken planlamak isteyenler

## Amaç
Öğrencilerin staj sürecine daha bilinçli ve hazırlıklı şekilde
girmesini sağlamak.
